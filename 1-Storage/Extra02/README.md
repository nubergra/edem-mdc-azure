# **Azure Storage - Extra Exercise 2: Using pgvector Extension for AI with Azure PostgreSQL**

## **Objective**

In this exercise, we'll use the **pgvector extension** with **Azure Database for PostgreSQL** to store and query vector embeddings generated by a free LLM model. We'll upload document embeddings to the database and perform similarity queries using **LangChain** in Python.

## **Steps**

1. Enable the **pgvector extension** in your Azure PostgreSQL server instance.
2. Install and configure **pgvector** in your PostgreSQL database.
3. Create a **Python script** to generate embeddings from documents and store the embeddings in PostgreSQL with **pgvector**.
5. **Query Data** and run vector similarity queries in PostgreSQL to retrieve relevant documents.
